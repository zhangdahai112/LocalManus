# System Execution Skill

This skill enables Claude to execute Python code and shell commands in a secure environment.

## Capabilities

- Execute Python code snippets
- Run shell commands
- Safely interact with the underlying system

## Usage

When you need to run Python code or execute system commands, Claude will automatically use this skill. Simply provide the code or command you want to execute.

## Technical Implementation

This skill integrates with:
- AgentScope Python execution tools
- Shell command execution utilities
- Security measures to prevent unsafe operations

The skill follows Claude's standard tool protocol for safe system interaction.
