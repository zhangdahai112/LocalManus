# Web Search Skill

This skill enables Claude to perform web searches and scrape content from websites.

## Capabilities

- Search the web using DuckDuckGo
- Scrape text content from web pages
- Extract relevant information from search results

## Usage

When you need to search the web or extract information from a website, Claude will automatically use this skill. Simply ask for information or specify a URL to scrape.

## Technical Implementation

This skill integrates with:
- DuckDuckGo search API for web searches
- BeautifulSoup for web scraping
- Custom Python functions for processing results

The skill follows Claude's standard tool protocol for safe and effective web interaction.
